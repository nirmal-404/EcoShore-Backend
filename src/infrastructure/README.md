# Infrastructure Layer

This layer contains implementations of external concerns like databases, authentication, and logging.

## Database
- **Models**: Database schemas (Mongoose, Sequelize, etc.)
- **Repositories**: Data access layer that abstracts database operations

## Auth
Authentication and authorization implementations.
- OAuth providers (Google, Facebook, etc.)
- JWT token management
- Session management

## Logging
Logging implementations and configurations.
- Application logging
- Error logging
- Audit logs
